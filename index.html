Criptografía RSA

Seguridad digital basada en la criptografía asimétrica

¿Qué es RSA?

RSA es un sistema criptográfico asimétrico que se basa en la dificultad de factorizar números grandes en sus factores primos. Permite cifrar y descifrar información de manera segura usando un par de claves: pública y privada.

Metodología RSA

Generación de claves

Se eligen dos números primos grandes p y q, se calcula n = p*q y el indicador de euler φ(n). Luego se elige e y d.

Cifrado

Se utiliza la clave pública (n, e) para convertir el mensaje en un número cifrado mediante exponenciación modular.

Descifrado

Se utiliza la clave privada (n, d) para recuperar el mensaje original del cifrado.
